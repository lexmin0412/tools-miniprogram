# tools-miniprogram

本项目基于 [taro3-react-template](https://github.com/lexmin0412/taro3-react-template) 构建。

## 技术栈

- Taro
- React Hooks
- TypeScript
- SCSS

## 支持特性

- 🏠 基于 Taro3
- 📦 支持 React
- 🐑 CSS 预处理器( SCSS )
- 🥣 完全使用 TypeScript 开发
- 🔛 企业级的 request 类及拦截器封装
- 👮 `eslint`+`stylelint`+`prettier`+`commitlint`+`editorConfig` 实现的无死角代码规范体系
